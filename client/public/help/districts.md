# 建筑说明

## 建筑牌信息

每张建筑牌包含：

- 名称
- 费用
- 颜色
- 分数
- 描述

## 建造方式

玩家在自己的行动阶段，可以从手牌中选择建筑进行建造。

建造时：

- 服务端检查是否轮到该玩家。
- 服务端检查该建筑是否在玩家手牌中。
- 服务端检查金币是否足够。
- 服务端扣除金币。
- 建筑从手牌进入城市区。

## 建筑颜色

建筑颜色会影响部分角色技能。

当前测试版中，商人会根据绿色建筑数量获得额外金币。

## 建筑分数

当前测试版中，建筑分主要来自建筑自身分值。

游戏结束时：

- 系统统计每名玩家城市中的建筑分。
- 满足结束条件的玩家可能获得奖励分。
- 建筑分和奖励分合计为总分。

## 当前测试版限制

- 暂时没有完整建筑特殊效果。
- 暂时没有正式卡牌美术。
- 建筑牌数据来自服务端 JSON 配置。
- 前端不能看到建筑牌堆顺序。
